# Arduino Basics

## Overview

This repository contains beginner-friendly Arduino projects covering the fundamentals of embedded systems and electronics.

## Projects Included

- LED Blink
- Push Button
- Digital Input
- Digital Output
- Analog Read
- PWM (Analog Write)
- Serial Monitor

## Hardware Used

- Arduino Uno
- Breadboard
- LEDs
- Push Button
- Resistors
- Jumper Wires
- Potentiometer

## Software

- Arduino IDE

## Skills Learned

- Digital I/O
- Analog I/O
- PWM
- Serial Communication
- Embedded Programming
